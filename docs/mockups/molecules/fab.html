FAB · Floating action button

Primary capture action on mobile. Tap to scan, long-press to open mode menu (Recibo / Lote / Estado de cuenta). Used on Dashboard, History, Insights — anywhere the user needs one-tap to capture.

Sizes

sm (44px)
md (default 56px)
lg (64px)

States

default
hover (lifted)
active (pressed)
focus-ring

Long-press · scan-mode menu

closed
open · 3 modes

Composition

  • Atoms used (one level down): none — primitive molecule, composes raw button + menu
  • Used by screens: Dashboard mobile · History mobile · Insights mobile · any capture-eligible mobile screen
  • State matrix: default · hover · active · focus-visible · open (menu visible)
  • Interaction: tap → default action (scan recibo). Long-press 500ms → opens menu. Tap-outside → closes menu.
  • A11y: aria-haspopup="menu" + aria-expanded on FAB. Menu uses role="menu" + role="menuitem". Always provide aria-label — icon alone is not accessible.
  • Position: bottom-right corner, inset 16px from screen edges, above bottom-nav.
