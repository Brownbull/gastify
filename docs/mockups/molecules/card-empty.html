Card · Empty

Empty-state placeholder. Used when a list, history, or group has no entries yet. Always pairs with a CTA to seed the first entry. Dashed border distinguishes from real cards.

Variants

history empty
📭

Aún no hay gastos

Escanea tu primer recibo o agrega un gasto manual para empezar a ver tu historial.

filter empty (no results match)
🔍

Sin resultados

Ningún gasto coincide con los filtros aplicados. Probá ajustando el rango de fechas o quitando una categoría.

group empty
👥

Tu grupo está vacío

Invitá a otros miembros con un enlace o cargá el primer gasto compartido.

Composition

  • Atoms used (one level down): button (btn-primary, btn-ghost, btn-secondary)
  • Used by screens: History · Insights · Group home · Settings → Datos
  • State matrix: single state — empty is a terminal state, not interactive
  • Tone: friendly + actionable. Always include at least one CTA. Avoid blame ("you have nothing") — frame as forward motion ("get started").

Usage

<article class="card card-empty">
  <div class="card-empty-icon">📭</div>
  <p class="card-empty-title">Aún no hay gastos</p>
  <p class="card-empty-body">Escanea tu primer recibo...</p>
  <button class="btn btn-primary btn-sm">Escanear</button>
</article>
