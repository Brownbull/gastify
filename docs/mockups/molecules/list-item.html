Composition

  • Atoms used (one level down): none — primitive molecule, composes raw text + check indicator
  • Used by screens: Settings · Sheet menus · Group member list · History selection mode · Insights · Items
  • State matrix: default · hover · selected · pressed · swipe-revealed (mobile only) · disabled
  • A11y: selection mode uses role="checkbox" + aria-checked + tabindex. Plain navigation uses <a href>. Swipe actions need keyboard fallback (long-press menu).
  • Swipe direction: right-to-left reveals destructive (delete) on right. Some flows use left-to-right for primary action (archive).
