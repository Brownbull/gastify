Composition

  • Atoms used (one level down): button (close + footer actions)
  • Composes: card-transaction · filter-chip · list-item
  • Used by screens: History desktop (tx detail) · Statement review · Group settings · Batch review
  • State matrix: opening · default (open) · closing
  • A11y: role="dialog" + aria-labelledby. ESC closes. Tap-scrim closes. Focus trapped inside drawer.
  • Sizing: 420px max-width default. Pinnable on wide viewports (drawer can become permanent left rail when ≥ 1440px).
