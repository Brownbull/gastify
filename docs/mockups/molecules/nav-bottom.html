Nav · Bottom (mobile)

Mobile primary navigation — 5 fixed tabs with active-tab indicator (3px bar above icon). Mobile-only. Used on every authenticated mobile screen except modal flows. Active state communicated by primary color + indicator.

Composition

  • Atoms used (one level down): none — primitive molecule, composes raw anchor + icon + label
  • Used by screens: every authenticated mobile screen (Dashboard, History, Trends, Insights, Groups, Settings)
  • State matrix: default · hover · active (primary color + 3px indicator) · focus-visible · disabled
  • A11y: aria-label="Navegación principal" on nav · aria-current="page" on active item · 44×44 min hit-target
  • 5-slot rule: exactly 5 — never 4 or 6. Center slot is conventionally the primary action (Escanear here). If group context, "Grupos" replaces Tendencias for that screen set.
