Composition

  • Used by screens: Dashboard hero · Settings · Onboarding · post-action celebration moments
  • State matrix: default · dismissed (animates out + persists in localStorage)
  • Tone: opportunity-framed ("descubrí", "probá") — never coercive. Cohort variant must include privacy reassurance.
