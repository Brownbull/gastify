Composition

  • Atoms used (one level down): avatar (footer user chip)
  • Used by screens: every authenticated desktop screen (paired with nav-top)
  • State matrix: default · hover · active · focus-visible · is-collapsed (64px width, labels hidden, tooltips via title attr)
  • A11y: aria-label="Navegación lateral" on aside · aria-current="page" on active · collapsed state needs title for label fallback
  • Sections: max 3 sections — extending beyond loses scannability. Settings + secondary nav move to top-right user menu, not sidebar.
