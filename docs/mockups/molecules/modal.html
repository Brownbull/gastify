Modal

Centered overlay dialog. 5 variants — confirm, form, learning, error, credit. Shared shell (overlay → modal → header → body → footer). Three sizes: default 440px / md 560px / lg 720px. Animations slide-up + fade-in declared in molecules.css.

Destructive confirmation. Two buttons in footer (cancel + destructive).

Composition

  • Atoms used (one level down): button (multiple variants) · input · select · progress (credit variant)
  • Used by screens: Transaction editor (form) · Single-scan error (error) · Settings → Suscripción (credit) · onboarding (learning) · destructive actions (confirm)
  • Sizes: default 440px · .modal-md 560px · .modal-lg 720px
  • State matrix: default · loading (footer button .is-loading) · error (form variant w/ error-summary) · stacked footer (mobile / single CTA)
  • A11y: role="dialog" + aria-labelledby for confirm/form/learning/credit; role="alertdialog" for error. Trap focus inside modal. ESC closes (handled by tweaks.js).
