Nav · Top (desktop)

Desktop top bar — brand + global search + avatar/actions. 60px tall. Sticky to viewport top. Pairs with sidebar nav for full desktop chrome. Drops shadow on scroll (.is-elevated).

Composition

  • Atoms used (one level down): button (btn-icon, btn-primary, btn-ghost) · avatar · input (search)
  • Used by screens: every authenticated desktop screen (alongside sidebar) · public auth flows (minimal variant)
  • State matrix: default · is-elevated (scroll shadow) · minimal (no search, no actions cluster)
  • A11y: use <header> + role="banner" implicit · skip-link "Saltar al contenido" lives just before this
  • Search: uses filter-search molecule pattern — same shape as filter-strip search field
