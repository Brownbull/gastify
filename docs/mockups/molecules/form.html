Composition

  • Atoms used (one level down): input · select · button · chip (radio-as-chip pattern)
  • Used by screens: Manual entry · Group create · Statement upload · Settings → Preferencias · Jurisdiction consent
  • State matrix: default · valid · invalid (per field) · submitting · with-error-summary
  • A11y: stepper as role="navigation" with aria-current="step" on active. Conditional sections use aria-live="polite". Error summary uses role="alert" + jump links to invalid fields.
  • Conditional reveal: always animate (fade + slide) when toggling. Don't reflow without transition — disorienting.
