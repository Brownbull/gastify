Sheet · Bottom sheet

Mobile pattern. Slides up from bottom. Drag-handle at top hints draggability. Used for filter selection, scan-mode picker, transaction edit on mobile, share menu. Modal-equivalent for narrow viewports.

Composition

  • Atoms used (one level down): button (footer actions)
  • Composes: list-item · filter-chip (sub-component reuse)
  • Used by screens: History (filter / actions) · Single-scan mobile (mode picker) · Transaction editor mobile · Group share
  • State matrix: default · dragging (drag-handle visible feedback) · dismissing (slides down)
  • A11y: role="dialog" + aria-labelledby. Drag handle is decorative (aria-hidden). Tap-on-scrim closes.
  • Desktop fallback: on viewports ≥ 768px, sheet upgrades to centered modal. Same content, different chrome.
