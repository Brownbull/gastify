Filters

Filter strip — chip-based filters with optional count badge + caret for popover. Scrolls horizontally on overflow (mobile). Variants: date, category, amount range, tag, search. Used on History, Insights, Group transactions.

Composition

  • Atoms used (one level down): chip (close to filter-chip but with active state) · input (search) · button (clear)
  • Used by screens: History · Insights · Group transactions · Items
  • State matrix: default · hover · active · with-count · disabled (gray + no caret)
  • Mobile: strip becomes horizontally scrollable (overflow-x: auto; flex-wrap: nowrap). Search collapses to icon-only with chip-popover.
  • A11y: filter-chip uses aria-pressed when active. Caret is decorative (aria-hidden). Count badge needs aria-label="N filtros activos".

Usage

<div class="filter-strip">
  <button class="filter-chip is-active" aria-pressed="true">
    <span>📅</span>
    <span>Este mes</span>
    <span class="filter-chip-count" aria-label="3 filtros activos">3</span>
    <span class="filter-chip-caret" aria-hidden="true">▾</span>
  </button>
</div>
