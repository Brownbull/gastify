Banner

Full-width inline notice. Persistent (unlike toasts). Variants: info, warning, error, offline. Optional dismiss + inline action button. Used at top of screens or inline within content for context-bound notices.

Variants

info
i Tu sincronización terminó hace 2 minutos. 128 gastos al día.
warning
! Te quedan 3 escaneos este mes. Pasá a Plan Pro o esperá hasta el 1 de mayo.
error
⚠ No pudimos cargar tu estado de cuenta. El PDF parece estar protegido o dañado. Probá subirlo de nuevo o cargá los gastos a mano.
offline (full-width, edge-bleed)
⊘ Sin conexión. Tus gastos se guardan localmente y se sincronizarán cuando vuelvas a estar en línea.

Composition

  • Atoms used (one level down): button (btn-primary, btn-ghost, btn-icon for dismiss)
  • Used by screens: Dashboard top (sync status, credit warning) · Statement upload (error) · global offline indicator
  • State matrix: default · with-action · dismissible · sticky-header (offline)
  • A11y: info/warning/offline use role="status" (polite); error uses role="alert" (assertive). Always include the meaning in text — never rely on color alone.
  • Offline variant: edge-to-edge (no border-radius, no side-borders) — sits at the very top of the viewport above any nav.

Usage

<div class="banner is-warning" role="status">
  <span class="banner-icon">!</span>
  <span class="banner-body">Te quedan <strong>3 escaneos</strong> este mes.</span>
  <div class="banner-actions">
    <button class="btn btn-primary btn-sm">Ver planes</button>
    <button class="btn btn-icon btn-sm" aria-label="Cerrar">×</button>
  </div>
</div>
