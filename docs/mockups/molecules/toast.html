Toast

Flash notification. Slides in from bottom-right (desktop) or top (mobile, override). Auto-dismisses after 5s by default. Stack vertically — newest on top. 4 variants: success, info, warning, error.

Variants

✓

Gasto guardado

Jumbo Costanera Center · $24.890 agregado a Mercado.

i

Sincronizando

Subiendo 3 gastos guardados sin conexión.

!

Te quedan 3 escaneos este mes

Pasá a Plan Pro para escaneos ilimitados.

×

Error al guardar

No hay conexión. Tu gasto se guardó localmente y se sincronizará después.

States

✓

Default state

Slid in, awaiting auto-dismiss timer.

i

Dismissing state

Slides out + fades. Animation locked in molecules.css.

Composition

  • Atoms used (one level down): none — primitive molecule, composes raw elements with molecules.css patterns
  • Used by screens: any post-action confirmation (save, delete, sync, batch upload)
  • State matrix: entering · default · dismissing
  • Auto-dismiss: 5s default · 8s for warning · sticky for error (manual close required)
  • A11y: success/info/warning use role="status" (aria-live=polite) · error uses role="alert" (aria-live=assertive)
