Desktop Web — Ítems

  • Full data table (8 cols) vs legacy mobile card stack — desktop gets dense tabular view of 240+ unique items
  • Row structure: icon / name+L4 chip + brand brands / top-store + frequency / count / avg price + min / total + % of month / trend delta / chevron
  • Grouped by L3 Familia (collapsible) — group header shows total + item count + distinct L4 count
  • View toggle: Agrupado (default, this mockup) / Plano (flat list of every tx-line, not aggregated — swap data-view)
  • Sort: Total / Compras / Precio promedio / Alfabético / Tendencia. Default = Total desc
  • Filter strip drives the universe — filtering L3=Alimentos Frescos + Envasados scopes table (2 seleccionados shown in Aplicar button)
  • Inflation flag: trend ≥+15% auto-flags row as .flagged (danger-soft bg, error amount color) — ties to Insights "actionable" type
  • Click row → drill to Transaction Editor filtered to transactions containing that item
