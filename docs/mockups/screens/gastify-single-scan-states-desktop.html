U5 · Stream en vivo

Ves el progreso en tiempo real vía SSE — sin "dead air" durante los 8 segundos de extracción. Cada paso del pipeline emite eventos mientras procesa.

Desktop Web — Single Scan (4 states)

  • Demo state tabs allow cycling through idle → uploading → processing → ready (real app uses live state machine, not clickable tabs)
  • State 1 idle: drag-drop zone + camera (webcam) + paste URL SII. Desktop gets Ctrl+V paste shortcut; mobile gets camera-first
  • State 2 uploading: spinner + progress bar + filename + ETA
  • State 3 processing: receipt thumb with scanline animation + pipeline step log (U5 streaming: each step emits SSE event; tokens/latency/cost update in telemetry rail)
  • State 4 ready: extracted data shown inline; actions are Descartar / Editar ítems (→ Tx Editor) / Quicksave (⚡)
  • Right rail: telemetry (U8 measure) + tips + U5 callout
  • Mobile equivalent: fullscreen camera with corner guides, single CTA per state, tips as sheet below
