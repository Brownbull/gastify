  • Hero gradient card matches Insights + Dashboard patterns — Normal (primary→success), Pro (primary→accent), Mono (ink→gray)
  • Items auto-grouped by L4 Categoría after extraction (category group header shows count + parent L3 + subtotal)

  • Shortcuts: ⌘+Enter = Quicksave · Esc = discard · ⌘+E = open Tx Editor
  • USD shadow auto-computed at tx save time using that date's BCCh rate — visible in header (CLP $23.490 · USD $24.75) + moneda field
  • Reconciliación check: if subtotal + IVA ≠ total shown on receipt, block Quicksave and force Tx Editor hard-lock mode
  • Mobile equivalent: full-screen card stack (hero → items accordion → metadata fields → fixed-bottom CTA), no rail
