Desktop Web — Settings Platform Notes

  • Nested-subnav pattern: 9-entry left list + right subview form. Mobile collapses to horizontal tab row + stacked subview
  • Right rail dropped — settings form benefits from wider form area vs 340px rail
  • Active subview scrolls to anchor on link click; subnav stays sticky
  • Save bar sticky at bottom: 20px — appears only when form is dirty (any input modified)
  • Keyboard: ⌘S saves (intent, not wired); Esc discards and restores last-saved state
  • Theme picker: live preview — clicking a theme option also flips body[data-theme] so user sees the change in-page
  • Subview anchors preserve deep-linking: gastify-settings-desktop.html#preferencias opens directly to Preferencias
  • Native-only settings (Hápticos) marked explicitly — web ignores but still persists per-user (synced across devices)
