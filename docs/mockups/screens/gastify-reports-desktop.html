Desktop Web — Reportes

  • 4 accordion groups (Semanal / Mensual / Anual / Personalizado) in a 2-col grid; each opens/closes independently; state persisted in localStorage client-side in P2
  • Split panel identical to Insights: list 1fr + drawer 420px sticky — click a row to load its detail
  • Status pills: ready (green) / scheduled (amber "En curso") / draft (gray)
  • Filter strip scopes ALL groups — filtering to "Supermercados L1" filters reports to L1-scoped ones and rewrites Personalizado suggestions
  • Drawer includes L1 rubro breakdown mini-bars + tied insights (link to gastify-insights-desktop.html)
  • Export actions: PDF (full report) + CSV (raw data) + Delete (destructive, confirms in-line)
  • "Nuevo reporte" → opens builder modal (not in this mockup); "Programar" → recurring-report config
