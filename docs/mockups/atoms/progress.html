Progress

Determinate progress indicator — linear bar or circular ring. Use for upload %, budget consumption, credit depletion, scan-processing steps. Transitions width smoothly; respects prefers-reduced-motion.

Linear — sizes

thin 45%
default 65%
thick 85%

Linear — semantic colors

default (primary) 50%
success 100%
warning 80%
error 95%

Linear — value stages

0% 0%
25% 25%
50% 50%
75% 75%
100% 100%

Circular

0% 0%
25% 25%
60% 60%
100% 100%
lg · 72 72%

Usage

<!-- Linear -->
<span class="progress" role="progressbar"
      aria-valuenow="65" aria-valuemin="0" aria-valuemax="100">
  <span class="progress__bar" style="width: 65%;"></span>
</span>

<!-- Semantic (warning/error when near limit) -->
<span class="progress progress-warning">
  <span class="progress__bar" style="width: 80%;"></span>
</span>

<!-- Circular — --value is the percent 0-100 -->
<span class="progress-circle" style="--value: 60;">
  <span class="progress-circle__label">60%</span>
</span>
