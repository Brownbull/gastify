Pill

Toggle segment for timeframe selectors, filter groups, status switches. One-of-N selection pattern (pair with a group container). Rounded-full radius distinguishes it from chips (context: pills toggle state, chips filter values).

Variants

default
active
with count
active + count

Group — timeframe selector

Group — with counts

States

default
hover
active
disabled

Usage

<!-- Simple pill -->
<button class="pill">Semana</button>
<button class="pill is-active">Mes</button>

<!-- Pill with count -->
<button class="pill">
  Supermercado
  <span class="pill__count">12</span>
</button>

<!-- Group (timeframe, filters, segmented control) -->
<div role="tablist">
  <button class="pill" role="tab">Año</button>
  <button class="pill is-active" role="tab" aria-selected="true">Mes</button>
</div>
