Pill

Toggle segment for timeframe selectors, filter groups, status switches. One-of-N selection pattern (pair with a group container). Rounded-full radius distinguishes it from chips (context: pills toggle state, chips filter values).

active
with count
active + count

LEGACY BoletApp 2024 reference — live

Live extraction from BoletApp staging via npm run extract:legacy. Source: viewmode pills on /tendencias (icon-based segmented control with 4 options: 🏪 stores · 🛒 categories · 📦 product groups · 🏷️ product categories). Note: BoletApp's pill is icon-only emoji; gastify pills are text + optional count.

Source: live · BoletApp staging · /tendencias

Usage

<!-- Simple pill -->
<button class="pill">Semana</button>
<button class="pill is-active">Mes</button>

<!-- Pill with count -->
<button class="pill">
  Supermercado
  <span class="pill__count">12</span>
</button>

<!-- Group (segmented control — atom layer)
     P3 molecule layer wraps in role="tablist" + aria-controls when paired with panels -->
<div>
  <button class="pill" type="button">Año</button>
  <button class="pill is-active" type="button" aria-pressed="true">Mes</button>
</div>

Used by molecules (one level up)

none directly — pills are consumed by screens (P5-P12) for filter labels, status chips, and inline metadata. The molecule layer uses chip for selection-style components instead.
