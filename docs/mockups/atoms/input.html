Input

Form input with label. Variants: text · password · email · number · search. Standalone + with prefix/suffix affix. States: default · filled · focused · disabled · error.

Variants

⌕
$

States

Email no es válido
La contraseña se valida al enviar

Usage

<div class="field-group">
  <label class="field-label" for="merchant">Comercio</label>
  <input id="merchant" type="text" class="field-input" placeholder="Jumbo">
</div>

<!-- With affix (icon / currency prefix) -->
<div class="field-affix">
  <span class="field-affix__icon">⌕</span>
  <input type="search" class="field-input" placeholder="Buscar">
</div>

<!-- Error state -->
<input type="email" class="field-input field-error" value="tu@">
<span class="field-error-msg">Email no es válido</span>
