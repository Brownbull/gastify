4e · “Selection = depth” rule

A consistent rule for any pick-one control: unselected = flat & soft (recedes), selected = ink border + little shadow (pops). Today selection is mostly just a fill change.

Our big numbers/titles currently use Baloo 2 (rounded, friendly). Gustify uses Outfit-extrabold everywhere (cleaner, more neutral). Same content, both fonts:
