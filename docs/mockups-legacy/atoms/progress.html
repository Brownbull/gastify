Progress

Linear progress bar — track + fill, optional indeterminate shimmer. The frontend uses Tailwind bg-amber-500 h-2 rounded-full transition-all with inline width percentages; we translate to .progress + .progress__bar with style="width: 73%" for visual fidelity. ARIA contract: determinate bars require role="progressbar" + aria-valuenow + aria-valuemin + aria-valuemax + aria-label. Indeterminate shimmer uses role="progressbar" + aria-busy="true" (no valuenow). L2 consumers (card-stat, profile-dropdown, processing-overlay) inherit this contract.

Source: BatchProcessingOverlay.tsx (batch progress 12/50), Nav.tsx (credit usage bar in profile dropdown).
