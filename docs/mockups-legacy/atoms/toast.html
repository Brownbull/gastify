Toast

Notification card — atom-level visual only. The queue + auto-dismiss layer (max-3 FIFO eviction, 3s fade-out) lives in frontend/src/shared/hooks/useToast.ts and is molecule territory at L2.

Source: shared/ui/Toast.tsx — canonical implementation. Hook layer at shared/hooks/useToast.ts (queue/auto-dismiss; molecule).

Anatomy

Position (live): position: fixed; bottom: calc(64px + safe-bottom); left: 50%; transform: translateX(-50%); — reserved space for nav-bottom on mobile. The mockup demos render the toast inline to make all four variants visible simultaneously.

Animation: animate-fade-in on mount (200ms ease-out, opacity + translateY 8px → 0). Auto-dismiss after 3000ms is hook responsibility (out of atom scope).

ARIA: role="status" + aria-live="polite" for success / info / warning. role="alert" + aria-live="assertive" for error per WCAG.

Populated during Phase L2 — molecule extraction. Anticipated consumer: molecules/toast-system.html wrapping the queue + ToastProvider + auto-dismiss layer (canonical L2 implementation referenced from shared/hooks/useToast.ts).
