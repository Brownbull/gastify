Authoring conventions

  • Inline <svg viewBox="0 0 24 24" aria-hidden="true"> wrapped in <span class="icon icon--lg"> — never <img src="..."> for app icons (color-flow + bundling reasons).
  • Stroke colors via currentColor — never hard-code stroke="#...". Set color on the parent.
  • Decorative icons get aria-hidden="true"; meaningful icons get aria-label on the parent button (not the icon).
  • Emoji icons (.icon--emoji) get role="img" + aria-label="<key>" on the span.
  • Sizes: sm=14, default=16, lg=20, xl=24, 2xl=32 — match lucide's standard sizes.
