## Methodology

For each molecule:
1. Grep the source file by `import.*<Source>` and `from.*<Source>` across `frontend/src/**/*.{ts,tsx}` (exclude the source file itself + `node_modules`)
2. For each consuming file, walk up the import graph until a top-level `*View.tsx` is reached
3. Cross-check against the route map in `viewRenderers.tsx`
4. For molecules with raw HTML attributes (e.g. `role="tablist"` not via a named component), grep the attribute directly across `views/` paths
5. Mark dead exports — components that appear in `index.ts` re-exports but have zero non-self consumers

**Refresh cadence:** re-run when a new view ships or a feature folder restructures. The truth-of-source pattern means snapshots rot fast; this doc is regenerated, not patched.
